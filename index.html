Как Выбрать для себя героя?

Для этого есть 3 способа:

  1. Посмотреть стиримы по доте и выбрать героя который тебе нравится по геймплею.
  2. Зайти в доте в режим Single Draft, это режим где тебе доётся выбор из 3 рандомных героев, спустя десятка игр ты найдёшь себе героя по вкусу!
  3. Попробовать поиграть на представленных “Лучших героев для новичков” на нашем сайте!
